# JobApplication Website

The **JobApplication Website** is a platform designed to help users easily apply for jobs by submitting their resumes and personal details. It allows employers to manage job postings, view applications, and track candidate progress.

## About

The JobApplication Website allows job seekers to:
- Browse job listings and apply directly.
- Upload resumes, cover letters, and other documents.
- Track application statuses.

Employers can:
- Post available job openings.
- Review applications and manage candidates.

This project is built using **Razor Pages**, a simple and effective way to structure a web application with .NET.

## Features

Key features of the JobApplication Website:
- **User Accounts**: Job seekers can register and create profiles.
- **Job Listings**: Employers can post job openings with specific details.
- **Application Submission**: Job seekers can apply for jobs and upload necessary documents.
- **Admin Panel**: Admins can manage users, jobs, and applications.

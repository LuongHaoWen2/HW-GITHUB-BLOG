## About

The **Car Rental Management System** provides a seamless way to manage car rentals for both customers and administrators.

Key functionalities include:
- **Customer Booking**: Customers can view available cars, check prices, and book rentals.
- **Admin Dashboard**: Admins can manage vehicle inventory, view bookings, and generate rental reports.
- **SQL Database Integration**: The backend uses a SQL database to store vehicle, booking, and customer information.

This application is built using **Blazor** for the front-end and **SQL** for the back-end, providing a full-stack web application.

## Features

Key features of the Car Rental Management System:
- **Customer Features**:
  - Browse available cars.
  - Check rental prices and car details.
  - Book and cancel rentals.
  - View rental history.

- **Admin Features**:
  - Add, edit, or remove vehicles from the inventory.
  - Manage customer bookings and view detailed reports.
  - Track rental status and car availability.

- **SQL Database**:
  - Store car inventory, customer information, and rental history.
  - Generate and manage bookings seamlessly with full CRUD (Create, Read, Update, Delete) functionality.
